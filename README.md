# «База даных студентов»

Для того, чтобы установить зависимости выполняем:

```sh
npm install
```

Для запуска сервера-api:

```
npm run start:server
```

Для запуска клиента:

```
npm run nodemon
npm run webpack-devserver
```

Для просмотра сервиса перейдите по ссылке:

```
http://localhost:3001/
```
